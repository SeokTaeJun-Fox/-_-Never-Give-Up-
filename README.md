## Never Give Up! (유니티 포트폴리오)

Unity를 사용하여 개발된 캐릭터 육성을 중심으로 한 성장형 RPG 게임입니다. 스토리와 진행은 "Never Give Up"이라는 주제를 중심으로 구성하여, 플레이어가 어떤 역경과 도전을 마주하더라도 끝까지 포기하지 않고 나아가는 흐름을 담도록 설계했습니다. 

### >> 코드 구조
* 옵저버 패턴/이벤트 기반 프로그래밍을 적용하여 느슨한 결합을 통해 객체 간 의존성을 낮추고, 유지보수와 확장을 쉽게 할 수 있도록 설계했습니다.
* 의존성 역전 원칙(DIP) 적용, 이벤트 채널(ScriptableObject)을 통해 객체 간 결합도를 낮추고 유연성 및 확장성을 높이도록 설계했습니다.
* 개방 폐쇄 원칙 적용으로 요구사항이 바뀌어도 새로운 자식 클래스를 추가해 기능을 확장할 수 있도록 설계했습니다.

